# SwiftShift — Packers & Movers Landing Page

A modern, fully responsive landing page built with **React + Vite + Tailwind CSS**.

---

## 🚀 Features

- **Responsive Design** — Mobile-first, looks great on all screen sizes
- **Smooth Animations** — Scroll-reveal, float animations, hover micro-interactions
- **SEO Optimized** — Semantic HTML5, Open Graph tags, JSON-LD structured data
- **Contact Form** — Name, phone, moving from/to, service selection, date picker
- **6 Services** — Home relocation, office shifting, vehicle transport, storage, international, furniture
- **4-Step Process** — Clear visual flow with animated step numbers
- **Testimonials** — 3 customer reviews with avatar, rating, and location
- **Clean Code** — Component-based architecture, easy to extend

---

## 📦 Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Google Fonts (Syne + DM Sans)

---

## 🏁 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🗂️ Project Structure

```
swiftshift/
├── index.html              # Entry HTML with SEO meta tags
├── package.json
├── vite.config.js
├── tailwind.config.js
├── postcss.config.js
└── src/
    ├── main.jsx            # React entry point
    ├── App.jsx             # Root component
    ├── index.css           # Global styles + Tailwind
    └── components/
        ├── Navbar.jsx      # Fixed nav with mobile hamburger
        ├── Hero.jsx        # Hero section with stats
        ├── Sections.jsx    # TrustBar, Services, HowItWorks, Testimonials
        ├── ContactForm.jsx # Contact form with validation
        └── Footer.jsx      # Footer with links
```
